# Lyricist Pro — AI Knowledge Base (EN)

## App Overview
Lyricist Pro is a professional-grade AI lyrics editor for songwriters and poets. It combines a structured lyric editor with AI generation, quantization, and musical prompt tools.

## Editor Modes
- **Text** — Free-form text editing, no structure constraints.
- **Section** — Structured editing with named sections (Verse, Chorus, Bridge, etc.). Each section holds multiple lines.
- **Markdown** — Markdown syntax for headings and inline formatting.
- **Phonetic** — IPA phonetic transcription view for pronunciation and rhythm analysis.

## Key Features
- **AI Generation** — Generate full songs or individual sections from a topic, mood, rhyme scheme, and target syllable count.
- **Suggestions** — Line-level AI suggestions: select a line, get 3 alternatives, click to apply.
- **Quantize** — Automatically adjusts all lines to a target syllable count while preserving meaning.
- **Analysis** — Deep structural analysis: rhyme quality, emotional arc, strengths, and musical suggestions.
- **Similarity** — Detect if your lyrics resemble existing songs in your library or on the web.
- **Versions** — Save named snapshots of your song; roll back at any time.
- **Translation / Adaptation** — Show a translation or lyrical adaptation of any line (toggle in Settings).

## Songwriting Panels
- **Left Panel** — Song metadata: title, topic, mood, rhyme scheme, target syllables, genre, tempo, instrumentation, rhythm, narrative.
- **Right Panel** — Song structure outline; click any section to scroll to it.
- **Suggestions Panel** — Opens when a line is selected in lyrics view.
- **Musical Tab** — Fill in genre, tempo, instrumentation, rhythm, and narrative for AI music-prompt generation.

## Sections & Structure
- Add sections with the **+** button; drag to reorder.
- Common section types: Verse, Pre-Chorus, Chorus, Bridge, Outro, Intro, Hook.
- Each section can be collapsed or expanded.
- Sections can be regenerated individually.

## Rhyme Schemes
- **AABB** — Couplet rhyme (lines 1–2 rhyme, lines 3–4 rhyme).
- **ABAB** — Alternating rhyme.
- **ABCB** — Ballad rhyme (only lines 2 and 4 rhyme).
- **Free** — No enforced rhyme pattern.

## Syllable Quantization
Set a target syllable count (e.g., 8) and press Quantize to align all lines. Use this to improve rhythmic consistency and singability.

## Keyboard Shortcuts
- **Ctrl/Cmd + Z** — Undo.
- **Ctrl/Cmd + Y** / **Ctrl/Cmd + Shift + Z** — Redo.
- **Ctrl/Cmd + F** — Open search & replace.
- **Escape** — Close modals and panels.

## Import / Export
- **Import** — Load `.txt`, `.md`, `.json`, `.docx`, `.odt` files or paste lyrics directly.
- **Export** — Save as `.txt`, `.md`, `.json`, or `.docx`.

## Library
Save songs to a local library for reuse and similarity reference. Access via the Export panel.

## Settings
- **Theme** — Dark / Light / System.
- **Audio Feedback** — Enable or disable UI sound cues.
- **UI Scale** — Small / Medium / Large font scaling.
- **Default Editor** — Choose the editor mode applied on new songs.
- **Translation / Adaptation** — Show or hide the translation feature column.

## Songwriting Tips
- Start with a strong hook (chorus) and build verses around it.
- Use a consistent syllable pattern across verses for singability.
- Vary rhyme density: dense rhymes for chorus, looser for verses.
- A bridge provides emotional contrast — change key, tempo feel, or perspective.
- Use the Mood field to guide the AI's emotional tone (e.g., "melancholic", "euphoric").
- Musical prompts (genre, tempo, instrumentation) are exported as AI-ready text for music generators.
